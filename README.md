A simple calendar widget

##Prereqs

 - Install Node 0.11.13 (I recommend you use nvm)
 - Install RethinkDB

##Install it

 - `npm install -g webpack`
 - `npm install`
 - `cd client && bower install`

##Hack on it

In four different terminal sessions run:

 - `rethinkdb`
 - `npm start`
 - `webpack -w`
 - `npm run lr`
